## 개요
이름부터 아주 직관적인, 주어진 값이 참인지 거짓인지를 판별하는 조건문이다.<br>
사용법 또한 간단하다.

## 이해를 돕는 그림


## 사용법
```swift
var food = "삼겹살"
If (food == "삼겹살") {
    print("소주로 달리자~")
} else if (food == "치킨") {
    print("시원한 맥주가 딱이지!")
} else {
    print("막걸리를 마")
}
```
